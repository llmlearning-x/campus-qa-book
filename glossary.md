# 术语表（中英对照）

| 中文术语 | 英文术语 | 说明 |
|---------|---------|------|
| 大模型 | LLM (Large Language Model) | 基于 Transformer 架构的大规模语言模型，如 GPT-4、Claude、文心一言 |
| 检索增强生成 | RAG (Retrieval-Augmented Generation) | 将外部知识检索与 LLM 生成相结合的技术 |
| 向量化 | Embedding | 将文本转化为高维向量的过程，用于语义相似度计算 |
| 向量库 | Vector Store / Vector Database | 专门存储和检索向量数据的数据库，如 FAISS、Chroma、Milvus |
| 文本切分 | Chunking / Text Splitting | 将长文档分割为小段落的处理过程 |
| 相似度检索 | Similarity Search | 基于向量余弦相似度或欧氏距离查找最相关片段 |
| TopK 匹配 | Top-K Retrieval | 返回与查询向量最相似的 K 个文档片段 |
| 提示词 | Prompt | 输入给大模型的指令和上下文文本 |
| 提示词工程 | Prompt Engineering | 设计和优化提示词以获得更好输出的技术 |
| 流式响应 | Streaming / SSE | 实时逐字输出模型响应，而非等待全部生成后返回 |
| 身份验证 | Authentication | 验证用户身份，如账号密码验证 |
| 授权 | Authorization | 控制用户可以访问哪些资源 |
| JWT | JSON Web Token | 一种紧凑的用于传递身份信息的 Token 标准 |
| CRUD | Create/Read/Update/Delete | 数据的增删改查基本操作 |
| REST | RESTful API | 基于 HTTP 协议的应用程序接口设计风格 |
| ORM | Object-Relational Mapping | 对象关系映射，将数据库表映射为代码对象 |
| 脚手架 | Scaffold | 快速创建项目骨架的工具 |
| 热重载 | Hot Reload | 代码修改后自动刷新，无需手动重启 |
| 跨域 | CORS (Cross-Origin Resource Sharing) | 浏览器安全策略，前后端分离时需要配置 |
| 软删除 | Soft Delete | 不真正删除数据，而是标记为已删除状态 |
