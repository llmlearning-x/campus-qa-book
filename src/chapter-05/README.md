# 第5章：RAG 核心与问答接口

> **核心问题**：怎样让 AI 基于上传的文档来回答问题？

---

这是全书最令人兴奋的一章。

前面四天，我们：
- Day 1~2：搭好了地基（框架、数据库、接口规范）
- Day 3：建好了门禁系统（用户认证、权限控制）
- Day 4：建好了「大脑的记忆」（知识库、向量化）

今天，我们要让这个系统真正「活」起来——实现从用户提问到 AI 回答的完整链路。

### RAG 问答完整流程

```
用户提问 ──► 前端发送请求 ──► FastAPI 接收
                              │
                              ▼
                         RAG 引擎检索
                              │
                              ▼
                    FAISS 向量相似度搜索
                              │
                              ▼
                    拼接 Prompt（问题+上下文）
                              │
                              ▼
                         发送给大模型
                              │
                              ▼
                    SSE 流式返回每个字
                              │
                              ▼
                    前端打字机效果显示
```
