# 第1章：项目启动与技术准备

> **核心问题**：一个 AI 问答系统，到底是由什么组成的？

---

在正式写第一行代码之前，我们需要搞清楚三件事：**我们要做什么**、**为什么这样做**、**怎么分工合作**。

这一章是整本书的地基。你会在这里建立起对整个系统的全局认知——从软件开发的基本流程，到 AI 问答系统的技术原理，再到把环境在你的电脑上搭起来。

## 本章学习目标

- [ ] 理解软件项目的基本开发流程
- [ ] 知道 LLM、Embedding、RAG 分别是什么意思
- [ ] 看懂系统架构图，能说出数据在各个组件间怎么流动
- [ ] 完成本地开发环境搭建，跑通环境验证脚本
