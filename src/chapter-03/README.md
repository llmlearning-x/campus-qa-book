# 第3章：用户系统与权限控制

> **核心问题**：谁能用这个系统？怎么控制访问权限？

---

几乎所有的 Web 应用都需要解决同一个问题：**识别用户身份，控制资源访问**。

今天我们要实现的用户系统，是整个校园问答助手的安全基础。没有它，任何人都能随意操作系统，包括删除知识库、查看所有人的问答记录。

本章涉及两个核心概念：
- **Authentication（认证）**：你是谁？（登录验证）
- **Authorization（授权）**：你能做什么？（权限控制）

## 本章学习目标

- [ ] 理解 JWT Token 认证机制
- [ ] 完成用户注册、登录接口
- [ ] 实现用户 CRUD（增删改查）
- [ ] 前端实现登录页、用户管理页
- [ ] 路由守卫：未登录自动跳转到登录页
